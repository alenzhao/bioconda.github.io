bioconductor-bayespeak
======================

|downloads|

This package is an implementation of the BayesPeak algorithm for peak-calling in ChIP-seq data.

======== ===========
Home     http://bioconductor.org/packages/release/bioc/html/BayesPeak.html
Versions 1.24.0
License  GPL (>= 2)
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-bayespeak
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`setup`), install with::

   conda install bioconductor-bayespeak

and update with::

   conda update bioconductor-bayespeak
