ndg-httpsclient
===============

|downloads|

Provides enhanced HTTPS support for httplib and urllib2 using PyOpenSSL

======== ===========
Home     https://github.com/cedadev/ndg_httpsclient/
Versions 0.4.2
License  BSD License
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/ndg-httpsclient

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`setup`), install with::

   conda install ndg-httpsclient

and update with::

   conda update ndg-httpsclient
